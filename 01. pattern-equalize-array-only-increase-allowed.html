Pattern: Equalize Array (Only Increase Allowed)

Problem Pattern

Given an array, all elements must be made equal. Allowed operation: • Increase an element by 1. • Decreasing is NOT allowed. Find the minimum number of operations.

Pattern

Greedy

🧠Intuition

Since decreasing is not allowed, the final value cannot be smaller than the current maximum element.

Any value greater than the maximum is valid, but choosing a larger target only increases the number of operations.

Therefore, the smallest valid target is the maximum element.

Approach

  1. Find the maximum element in the array.
  2. For every element, calculate: maxElement - currentElement
  3. Sum all these differences.
  4. The sum is the minimum operations required.
⏱️Time Complexity

O(N)

💾Space Complexity

O(1)

Recognition Keywords

• Make all elements equal • Only increase • Cannot decrease • Minimum operations

Pattern Recognition

Only Increase → Target = Maximum Element

Only Decrease → Target = Minimum Element

Increase/Decrease Both Allowed → Target = Median
